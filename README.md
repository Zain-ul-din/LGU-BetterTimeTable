## Getting Started

First, run the development server:

```bash
  ~ git clone https://github.com/Zain-ul-din/LGU-Timetable.git
  ~ cd LGU-Timetable
  ~ npm i yarn -g 
  ~ yarn
  ~ npm run dev
  # or
  ~ yarn dev
```
